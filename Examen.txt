GIT

¿Cual es la diferencia entre git y Github?

R= git es el software que utilizamos en nuestra computadora local como sistema de control de versiones, herramienta que utilizaremos en nuesgtra terminal. GitHub es el servicio de hosting de los proyectos que se desarrollan y los cuales se guardan y modifican a traves de este servicio.

¿En cuantos y cuales estados puede estar un archivo?
R= En 3  confirmado o (commited), modificado (modified), y preparado (staged).

¿Para que sirve el comando "git add" ?
R= Sirve para añadir los archivos nuevos o modificados al proyecto que se esta trabajando.

¿Cual es el comando para crear una rama en mi repositorio?
R= git branch "nombre"
